# Current version: 2.0.0.0
- Works as an IO driver, instead of a function block,
- All interaction are done via IO and Parameters, no function Block calls needed,
- This release supports;
  * 4 Di,
  * 4 Do,
  * 2 Ai,
  * 2 Ao,
  * Hardware Watchdog.
